# Site Clinica Garden
Estou finalizando o site institucional para a Clínica Garden onde são divulgados os tratamentos, depoimentos dos pacientes e o método da Dra.

# Cenário Atual
O cliente não possuí um site para sua empresa atualmente e divulga seu trabalho apenas no Instagram.

# Cenário desejado
Criar um site para divulgar produtos e serviços do cliente, o que irá permitir criar campanhas publicitárias nas principais plataformas.

# O que foi feito:
Está sendo criado um site em HTML CSS e Javascript com apoio de IA generativa para ser mais eficiente.
